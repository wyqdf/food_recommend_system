# 美食推荐系统 - 文档中心

> **版本**: v3.0
> **最后更新**: 2026-03-08

## 项目概况

- **后端**: Spring Boot 3.2.2 + MyBatis + MySQL
- **前端**: Vue 3 + Element Plus + Pinia + Axios
- **数据库**: MySQL 8.0 (600万+数据)
- **认证**: JWT Token
