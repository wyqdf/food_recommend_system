# 美食推荐系统 - 项目总结

## 项目概述
这是一个完整的美食推荐系统项目，包含前端、后端和管理后台三部分。

**技术栈：**
- 后端：Spring Boot 4.0.3 + MyBatis + MySQL
- 前端：Vue 3 + Element Plus + Pinia + Axios + Vue Router
- 数据库：MySQL 8.0

**数据库连接配置：**
- Host: localhost:3306
- User: root
- Password: 123456
- Database: food_recommend

### 表结构详解

#### 1. 映射表（重复文本用ID存储）

**分类表 (categories)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(50) NOT NULL UNIQUE
create_time DATETIME
```

**口味表 (tastes)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(50) NOT NULL UNIQUE  -- 原味、咸鲜、甜味等
create_time DATETIME
```

**工艺表 (techniques)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(50) NOT NULL UNIQUE  -- 炒、焖、煮、蒸等
create_time DATETIME
```

**耗时表 (time_costs)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(50) NOT NULL UNIQUE  -- 廿分钟、一小时等
create_time DATETIME
```

**难度表 (difficulties)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(50) NOT NULL UNIQUE  -- 普通、简单、高级等
create_time DATETIME
```

**食材表 (ingredients)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
name VARCHAR(100) NOT NULL UNIQUE
create_time DATETIME
```

#### 2. 核心业务表

**用户表 (users)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
old_id INT                    -- 原始用户ID
username VARCHAR(100)
nickname VARCHAR(100)
create_time DATETIME
UNIQUE KEY uk_old_id (old_id)
```

**菜谱表 (recipes)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
old_id INT                    -- 原始菜谱ID
title VARCHAR(200) NOT NULL
author VARCHAR(100)
author_uid VARCHAR(50)
description TEXT
tips TEXT
cookware VARCHAR(200)
taste_id INT                  -- 关联口味表
technique_id INT              -- 关联工艺表
time_cost_id INT              -- 关联耗时表
difficulty_id INT             -- 关联难度表
reply_count INT DEFAULT 0
like_count INT DEFAULT 0
rating_count INT DEFAULT 0
create_time DATETIME
update_time DATETIME
UNIQUE KEY uk_old_id (old_id)
```

**评论表 (comments)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
old_id INT                    -- 原始评论ID
recipe_id INT NOT NULL        -- 关联菜谱表
user_id INT                   -- 关联用户表
content TEXT
publish_time DATETIME
likes INT DEFAULT 0
is_reply TINYINT DEFAULT 0
reply_to_user VARCHAR(100)
device VARCHAR(200)
has_picture TINYINT DEFAULT 0
create_time DATETIME
```

#### 3. 关联表

**菜谱-分类关联表 (recipe_categories)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
recipe_id INT NOT NULL
category_id INT NOT NULL
UNIQUE KEY uk_recipe_category (recipe_id, category_id)
```

**菜谱-食材关联表 (recipe_ingredients)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
recipe_id INT NOT NULL
ingredient_id INT NOT NULL
ingredient_type ENUM('main', 'sub', 'seasoning')  -- 主料/辅料/调料
quantity VARCHAR(100)         -- 用量
```

**烹饪步骤表 (cooking_steps)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
recipe_id INT NOT NULL
step_number INT NOT NULL      -- 步骤序号
description TEXT NOT NULL     -- 步骤描述
```

#### 4. ID映射表

**用户ID映射表 (user_id_mapping)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
old_user_id INT NOT NULL      -- 原始用户ID
new_user_id INT NOT NULL      -- 新用户ID
UNIQUE KEY uk_old_user_id (old_user_id)
```

**菜谱ID映射表 (recipe_id_mapping)**
```sql
id INT PRIMARY KEY AUTO_INCREMENT
old_recipe_id INT NOT NULL    -- 原始菜谱ID
new_recipe_id INT NOT NULL    -- 新菜谱ID
UNIQUE KEY uk_old_recipe_id (old_recipe_id)
```

- Java：驼峰命名法 `recipeTitle`
