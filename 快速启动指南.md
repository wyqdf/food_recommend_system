# 快速启动指南

## 一、立即体验管理员功能

### 步骤 1：安装依赖
```bash
cd frontend
npm install
```

### 步骤 2：启动开发服务器
```bash
npm run dev
```

### 步骤 3：访问管理后台
打开浏览器访问：http://localhost:5173/admin/login

### 步骤 4：登录
- 用户名：admin

## 六、注意事项

1. **Mock 数据**: 当前所有数据都是模拟的，刷新后会重置
2. **登录状态**: 登录信息存储在 localStorage 中
3. **权限控制**: 访问管理页面需要登录状态
4. **浏览器**: 建议使用 Chrome 或 Edge 浏览器

## 七、常见问题

### Q: 无法访问页面？
A: 确保开发服务器已启动，访问 http://localhost:5173/admin/login

### Q: 登录后跳转？
A: 检查 localStorage 中是否有 admin_token

### Q: 图表不显示？
A: 确保已安装 ECharts: `npm install echarts@^5.4.3`

### Q: 样式不正常？
A: 确保已安装 Element Plus: `npm install element-plus@^2.5.0`
